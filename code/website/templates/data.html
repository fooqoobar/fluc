Your Data & Privacy


How do we store data?

All user data (user ID, username, email, access tokens, refresh tokens) is stored in an encrypted database and locally on your computer.

For additional security, we automatically delete your local data every week.


How do we use your data?

We use this data to save your login information on the website.


Scopes

When authorizing the bot, you might have noticed that it requires a few scopes.

  • Identify (identify)
  • Email (email)
  • Add user to servers (guilds.join)

identify is used to access your Discord username and user ID.

email is used to prevent spam and log activity. We do not send emails to users.

guilds.join is used to add you to the bot server. If the server is deleted by Discord, we can use this to add you back to a new one.


Lastly...

We do not sell user data. All data is stored in a private database that only three people have access to.

All sensitive data is encrypted!
